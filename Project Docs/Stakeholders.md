# Stakeholders

- Product Owner: defines acceptance, prioritizes use cases.
- Accountant: validates P&L and A&L reconciliation and uses module for audits.
- Warehouse Clerk: provides inventory context and barcode/serial usage.
- Purchasing Clerk: validates PO/Invoice mappings.
- Sales Clerk: verifies sales allocation and discounts.
- Bank Reconciliation Clerk: verifies deposit and fee attribution.
- Developers: implement FA hooks, UI and migrations.
- QA: verifies reports, performance and edge cases.
